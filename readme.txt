本项目主要用于对hayday的自动化操作、自动化排产，基于雷电模拟器海外版实现。
主要使用uiautomator2和python-opencv实现，少量使用pyautogui（在出现运行异常后重启模拟器，需要缩小游戏界面，此时模拟器会被置于前台）

核心文件为zhongzhi.py和zhuanshou.py，pictures文件夹，data文件夹，下面分别介绍其在本项目中的作用：
zhongzhi.py：自动化种植作物，适合小麦刷建材玩家，支持后台运行。
和zhuanshou.py：自动化交易商品（你就是倒爷!），适合低等级玩家快速积攒财富。在市场中低价买入商品，之后高价在货架中卖出。
pictures文件夹：存储图像对比的样本图片。
data文件夹：存储玩家游戏数据，比如等级、金币、各产品基本信息、各建筑生产状态等信息，主要用于排产。
即将会有的：auto_plan.py：用于自动化排产，自动收获产品，按照需求智能化放入原料开展生产，适合休闲玩家。

注：由于作者水平暂时有限（我会加油练习时长的！），在代码运行之前需要完成一些事情：
1.打开雷电模拟器，将模拟器分辨率设定为1600*900
2.在运行zhongzhi.py的时候，需要手动将布局和屏幕调整至调整到pictures/template/zhongzhi.png所示状态
3.在运行zhuanshou.py的时候，需要手动将布局和屏幕调整至调整到pictures/template/zhuanzhou.png所示状态
